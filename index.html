Hva er YumBite?

Din nye frokost- og snacksløsning på Bjerke VGS

YumBite er elevbedriften som gjør skoledagen din enklere og sunnere. Vi tilbyr:

  • Rask service: Ingen mer lange køer - få maten din på 1-2 minutter
  • Tidlig tilgang: Åpent fra 08:00-09:30 tirsdager og onsdager
  • Rimelige priser: Sunn mat som passer studentbudsjettet
  • Bærekraftige valg: Miljøvennlige produkter og emballasje

Hvorfor YumBite?

🕒 Spar verdifull tid

Slipp lange kantinekøer som tar 5-10 minutter av friminuttet ditt. Med YumBite får du maten raskt og kan faktisk rekke å spise den også!

🌅 Smart start på dagen

Kom til skolen litt tidligere og få en god frokost. Reduser stress og forsinkelser ved å spise på skolen istedenfor hjemme.

🎯 Tilpasset dine ønsker

Vi baserer vårt utvalg på deres tilbakemeldinger gjennom spørreundersøkelsen. Ditt valg påvirker hva vi tilbyr!

🌱 Bærekraftig valg

Vi velger leverandører og produkter med omhu for å minimere miljøpåvirkningen og støtte bærekraftig produksjon.
