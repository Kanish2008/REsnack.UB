Hva vi skal og hvordan vi hjelper dere

REsnack tilbyr praktiske og prisvennlige alternativer, tydelig merket for allergier. Teamets interne kontrakt er privat — denne siden viser det som er relevant for elever og samarbeidspartnere.

Tider og drift

Daglig drift: 08:00–09:30. Pakketeam møter 07:30. Vi sørger for trygg merking av ingredienser.

Spørreundersøkelse

Undersøkelsen er anonym. Vi samler ikke navn eller e-post. Svar lagres lokalt og kan eksporteres av daglig leder via admin-visningen.

Takk — svaret er mottatt.
